PSP Calendar Ver 0.3

■ 概要
PSP上で動作するカレンダーです。
ただし、ファームv1.0のPSPに限ります。

■ 使用方法
メモリースティックに /PSP/GAME/CALENDAR という形のフォルダを作り、
そこに同封の EBOOT.PBP Calendar.ini Holiday.ini を置いてください。
また、BACK.BMPを置くと、起動時に読み込んで背景として使用します。
必要にに応じて、スケジュールも作成してください。
終了は、HOMEボタンでメニューに戻って下さい。

■ 操作キー
カレンダー操作時
  十字キー     カーソル移動
  △ボタン     今日に移動
  ○ボタン     スケジュール詳細表示
  L/Rボタン    月移動

スケジュール詳細表示時
  ×ボタン     カレンダー表示へもどる
  十字キー     上下スクロール
  L/Rボタン    ページ Up/Down  


■設定ファイル
Calendar.ini  カレンダーの設定を記述します

[Env]         環境設定
  TZ=           TimeZoneを指定します(東京なら 9)
  Schedule=     スケジュールがあるフォルダーを指定します（記述がなければ、インストールフォルダー）

[WEEK]        曜日設定（カレンダー用）
  SUN=          日曜日のタイトル
  以下省略

[TIME]        曜日設定（時刻用）
  SUN=          日曜日のタイトル
  以下省略

Holiday.ini   休日、特別な日の設定（2010年まで設定していますが間違いがあるかもしれません）
[ALL]         年によって変化しない日の指定をします（例 毎年 1/1 元旦）
  月日=         休日設定  月2桁日2桁  (例 0115=成人の日)
  月日S=        休日設定  月2桁日2桁S (例 0701=誕生日)

[4桁の西暦]   その年にのみ適応する日の指定をします（例 体育の日）
  [ALL]と同様

■スケジュール
Scheduleのフォルダー下に、西暦4桁のフォルダー/月2桁のフォルダーを作成し
その中に、日付2桁.txtというテキストファイルを作成して、そのファイの中にスケジュールの内容を書いてください。
（例 2005年5月22日のスケジュール）
ms0:/PSP/GAME/CALENDAR/2005/05 フォルダーの中に 22.txt を置く。

■背景用ビットマップ
画像は以下のフォーマットを守ってください。
　・480x272x24bit の Windows bmp format (BottomUp)
　・ヘッダサイズが0x36byte（ファイルサイズが391,734byte）
通常のアプリで作成すれば、問題なく正しいファイルができるはずです。
